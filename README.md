## Selected work

### [Financial Risk News Classification](https://github.com/ezeriosk/financial-risk-news-analysis)

An interpretable NLP pipeline using TF-IDF, cosine similarity and transparent business rules to classify financial-risk news. Includes synthetic data, automated tests, an executed notebook and a monitoring dashboard.

### [Ticket Demand Forecasting](https://github.com/ezeriosk/ticket-demand-forecasting)

An operational analytics and capacity-planning case study using 60,000 synthetic tickets. Compares Holt-Winters and SARIMA forecasting while identifying demand peaks, SLA patterns and team bottlenecks.

### [Carsharing Pricing Analysis](https://github.com/ezeriosk/carsharing-pricing-analysis)

A product analytics investigation into upfront-pricing precision across 2,875 orders, connecting GPS quality, geography and application versions with price variation and customer impact.
